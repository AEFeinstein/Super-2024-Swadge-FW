# Digipicker

There's exactly one good mechanic in Bethesda's Starfield: Lockpicking. This mode acts like that minigame.

Each puzzle consists of 1-3 rings that have gaps that need to be filled. Keys are provided that contain the missing sections, but may need to be rotated into position. Additional key that are not used can be available. Only one set of keys per puzzle, so multi-ring puzzles require players to choose their keys carefully.

## Controls

- Up/Down: Choose key
- Left/Right: Rotate key
- A: Apply key
- B: Undo key
- Start: Menu

## Systems

- Menu (use default)
- Lockpicking
  - Initialization
    - Pre-load locked sections (Inner rings)
    - Pre-load keys
    - Random  puzzle generation
    - COlor selection
  - Gameplay
    - Rotating keys
    - Selecting keys
    - Timer
    - Key history/Undo
    - Multi-Ring
  - Other
    - Score
    - Menu
      - Color
      - Back out
